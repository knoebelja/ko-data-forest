## dev

make sure you have `node` LTS installed. i recommend using `nvm` for installing and managing `node` versions.

### setup
```bash
git clone git@github.com:knoebelja/ko-data-forest.git
cd ko-data-forest
npm install
```

### scripts
- `npm start`
- `npm run build`
- `npm run test`
