## dev

make sure you have `node` LTS installed. i recommend using [nvm](https://github.com/nvm-sh/nvm) for installing and managing different `node` environments.

### setup
```bash
git clone git@github.com:knoebelja/ko-data-forest.git
cd ko-data-forest
npm install
npm run build
```

### scripts
- `npm start`
- `npm run build`
- `npm run test`
